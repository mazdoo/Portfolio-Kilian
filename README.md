# Portfolio-Kilian

## Projet : analyse de texte en Python

Ce projet Python permet de :

- lire un fichier texte `.txt`
- lire un fichier `.json`
- nettoyer un texte
- compter la fréquence des mots
- afficher les 10 mots les plus fréquents

## Fichiers du projet

- `analyse_texte.py` : script principal
- `Entrainement.txt` : fichier texte analysé
- `infos.json` : fichier JSON utilisé dans le script

## Concepts Python utilisés

- `for`
- `while`
- `def`
- `return`
- `list`
- `dict`
- `tuple`
- lecture de fichier `.txt`
- lecture de fichier `.json`

## Lancer le projet

Dans le terminal :

```bash
python analyse_texte.py
